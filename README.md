Music app built with React.js. Data is taken from spotify api

https://eloquent-goldwasser-11658a.netlify.app/review

## Features
- Review albums, artists, playlists
- Music categories
- Search
- Music player
- Responsive design

## Build with

- [Redux.js (toolkit)](https://github.com/reduxjs/redux-toolkit "Redux.js (toolkit)")
- [Prop Types](https://github.com/facebook/prop-types "Prop Types")
- [React Router Dom](https://github.com/ReactTraining/react-router "React Router Dom")
- [Axios](https://github.com/axios/axios "Axios")
- [Styled Components](https://github.com/styled-components/styled-components "Styled Components")
